# 文档中心

这是个人搭建的导航与文档站，整合了日常使用的功能、工具与说明文档。

## 使用方法
- 在顶部搜索框输入文档名称，即可快速查找并打开对应页面
- 支持路径格式：`#文件夹/文档名`，会自动读取 `wiki/` 内对应的文档
- 找不到页面时会自动提示，可返回首页重新查找
- 所有文档支持 Markdown 格式，可直接查看排版内容

## 说明
本项目为个人自用导航页面，无多余依赖，轻量化、高性能、易扩展。

## 界面
搞了个彩色主题，界面简洁美观。
